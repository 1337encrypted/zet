# How memory works on a microcontroller

```
  Flash             RAM
 ----------       ----------
|          |     |          |
|          |     |          |
|   code   |     |   Data   |
|          |     |          |
|          |     |          |
 ----------       ----------

```
| FLASH | RAM |
| :-- | :-- |
| Non Volatile | Volatile |
| Static | Dynamic |
| Larger (eg 16kb) | Smaller (eg 512B) |
